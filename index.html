Centralized settings management for dotnet microservices.

Get Started with Fig

Easy to Integrate

Fig can be integrated into existing applications with just a few lines of code. Use the settings in your application just as any other class.

Supports Different Property Types

Specialized editors for strings, integers, bools, data tables, drop downs, secret settings and many more.

Live Reload

Updated values available to your application after they are applied in the web interface.

Offline Settings

Last loaded settings are available to clients even if the api is offline.

Client Management

See connected settings clients listed in real time and request restart of individual clients.

Audit Logging

All settings changes are logged and can be reviewed in the audit log.
