Keyboard Shortcuts

The fig web application supports keyboard shortcuts for common operations.

The following shortcuts are available:

ShortcutAction
Alt + sSave Client
Alt + aSave all Clients
Alt + eExpand all Settings
Alt + cCollapse all Settings
Alt + iCreate new instance
Alt + dShow Client Description
Alt + fFind Settings
